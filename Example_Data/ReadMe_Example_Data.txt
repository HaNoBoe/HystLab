This directory contains the data present in [1], the paper that introduces HystLab.

All data are in Princeton MircoMag VSM format.

If any data are used for other purposes the appropriate reference must be credited in the derivate work.
Reference details are given in the appropriate figure readme files.


References
[1] Paterson, G. A., X. Zhao, D. Heslop and M. Jackson (2018), Measuring, processing, and analyzing hysteresis data
